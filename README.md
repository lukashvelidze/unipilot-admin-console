# UniPilot Everywhere

UniPilot Everywhere is a comprehensive platform designed to help international students navigate the student visa application process.

## Features

- **Multiple Destinations**: Support for visa applications to numerous countries worldwide
- **Smart Checklists**: Personalized document checklists based on your visa type and situation
- **Expert Guidance**: Step-by-step guidance throughout your entire visa application journey
- **Track Progress**: Keep track of your application status and never miss a deadline

## Getting Started

Download the app from TestFlight: https://testflight.apple.com/join/13bcqnX4

## Development

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Setup

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd unipilot-admin-console

# Install dependencies
npm i

# Start the development server
npm run dev
```
